git status  -  показывает текущее положение вещей

git add . или git add имя_файла  -  добавляет к отслеживанию файлы

git commit -m "имя коммита можно на русском и даже с пробелами"  - делает коммит

git log показывает историю

git checkout хэш - переходит на какой-нибудь коммит (хэш которого указан)

git checkout -b "имя_ветки_на английском_без_пробелов"  переключается на новую ветку, одновременно создавая ее

git checkout "имя_ветки_на английском_без_пробелов"  просто переключается на новую ветку

git push origin master - пушит (отсылает коммиты в удаленный репозиторий)

git pull origin master - пуллит (скачивает коммиты из удаленного репозитория)
